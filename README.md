# Kargo Takip OOP

Basit bir JavaScript OOP projesi — kargo takip simülasyonu.

## İçerik
- `app/` : Uygulama giriş dosyaları
- `companies/` : Kargo şirketlerine ait modüller
- `core/` : Temel sınıflar ve mantık
- `services/` : Yardımcı servisler
- `utils/` : Genel yardımcı fonksiyonlar
- `index.html` : Proje ana sayfası

## Çalıştırma
1. Proje klasörünü açın:
```
cd "c:\\Users\\hamdi\\Desktop\\react_tutorial\\js\\js öğreniyorum\\Kargo Takip OOP"
```
2. Tarayıcıda `index.html` dosyasını açın veya basit bir sunucu çalıştırın:
```
python -m http.server 8000
# sonra tarayıcıda http://localhost:8000/ adresini açın
```

Alternatif: VS Code kullanıyorsanız `Live Server` eklentisi ile `index.html` dosyasını canlı görüntüleyebilirsiniz.
